- ### xLineCounter (Late 2022)
A lightweight yet powerful command-line utility designed for developers to quickly assess project scale and codebase metrics. Unlike basic line counters, xLineCounter intelligently distinguishes between source code and binary files, providing accurate line counts for text-based project files.  
Features include:
	- Recursive directory scanning – Count lines across entire project folders with a single drag-and-drop
	- Smart file filtering – Automatically ignores binary files, images, and compiled assets for accurate code metrics
	- Detailed breakdowns – Per-file and per-extension statistics with total line counts and summaries
	- Performance optimized – Handles large codebases with minimal memory footprint and fast execution
	- Progress feedback – Real-time progress indicator for large directory scans

Platforms: Windows
Languages: C#
Frameworks: .NET 8.0
